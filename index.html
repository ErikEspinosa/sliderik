How to use

Follow the next steps to add Sliderik! into your project:

1. Get Sliderik!

Download the repository from Github

2. Copy files

Locate the following files and copy in your project folder:

./styles/sliderik.css
./src/sliderik.js

3. Link to files

Add the next code in your index.html file:

							<link rel="stylesheet" href="./styles/sliderik.css" />
							<script src="./src/sliderik.js"></script>
